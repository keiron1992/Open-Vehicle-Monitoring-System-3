=======================
Maxus T90 EV

Vehicle Type: MT90EV

The Maxus T90 EV is an officially supported vehicle in OVMS. This module provides essential monitoring and control features for the all-electric pickup truck.

Support Overview

=========================== ==============
Function                    Support Status
=========================== ==============
Hardware                    OVMS v3 (or later)
Vehicle Cable               OBD-II to DB9 Data Cable for OVMS (1441200 right, or 1139300 left)
GSM Antenna                 1000500 Open Vehicles OVMS GSM Antenna (or any compatible antenna)
GPS Antenna                 1020200 Universal GPS Antenna (SMA Connector) (or any compatible antenna)
SOC Display                 Yes (State of Charge)
Range Display               Yes (Estimated Remaining Range)
GPS Location                Yes (from modem module GPS)
Speed Display               Yes
Odometer Display            Yes
Temperature Display         Yes (External Temp and Battery)
BMS v+t Display             Partial (High-Voltage Battery Temp and Voltage)
TPMS Display                No
Charge Status Display       Yes
Charge Interruption Alerts  Yes
Charge Control              No (Monitoring only)
Cabin Pre-heat/cool Control No
Lock/Unlock Vehicle         Yes
Valet Mode Control          No
Others                      No
=========================== ==============

.. note::
The support status reflects the latest stable firmware. Features are actively being developed and may improve in future releases.
